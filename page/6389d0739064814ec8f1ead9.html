Role: Data Engineer (Python/AWS) Take-Two, a leader in the interactive entertainment industry, is looking for a dynamic Data Engineer to join a team building a cloud-based data and analytics platform. The Data Engineer will design, build, and maintain reliable and scalable data pipelines. The ideal candidate is a strong Python developer equally comfortable with writing code and helping business teams understand data. He/she is a self-starter, comfortable with ambiguity, able to think big (while paying careful attention to detail) and enjoys working in a fast-paced environment. Role Responsibilities Design, develop and maintain high-capacity and scalable data pipelines. Design, develop and maintain API based ETL/ELT processes for raw data collection from multiple sources. Work with business teams and analysts to understand complex business rules and implement these for deriving insights. Contribute in maturing o the data engineering function including but not limited to data architecture, data quality, monitoring and alerting. Develop and support continuous integrations build and deployment processes using Jenkins, Docker, Git, etc. Qualifications 4+ years of Data Engineering experience of designing, developing and automating scalable ETL/ELT solutions that transform data into accurate and actionable business information 3+ years of hands-on experience in Python 3+ years of SQL experience using advanced SQL queries (analytical functions), writing and optimizing highly efficient queries 2+ years of experience working with AWS technologies stack like S3, EMR, etc. Experience building API’s (highly desirable) Experience in testing and monitoring data for anomalies and rectifying them Knowledge of software coding practices across the development lifecycle, including agile methodologies, coding standards, code reviews, source management, build processes, testing, and operations Bachelor’s degree or equivalent in an engineering or technical field such as Computer Science, Information Systems, Statistics, Engineering, or similar Preferred Skills Python (required) SQL (required) Understanding of EMR and related compute engines (required) Git (required) Build and deployment tools (for example Jenkins, Maven, SBT) Developing solutions using Docker What We Offer You: Great Company Culture.
