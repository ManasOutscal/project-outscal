Job Description

Are you looking for an opportunity to work remotely with a team of passionate game developers? Are you a reliable, effective, and quality focused Engineer that can level up a project and the team around you? If so, here is an opportunity at Stoic to iterate with design teams on our ambitious new title! As a JavaScript UI Developer, you will work closely with all disciplines to build immersive UI experiences. Mastering CSS, JavaScript, and visual editing pipelines, you will be able to craft interfaces for players that will help tell the story of our game and provide easy to understand interactions. What You Will Do Build and maintain in-world and full screen user interfaces using web technologies Integrate live data and events from Unreal Engine into our custom JavaScript UI framework Ideate and build upon the core web framework for performant and adaptive screens Layout screens, create new features, and apply JavaScript in our visual UI editing pipeline Be part of a collaborative User Experience team that crafts easy to use features and detailed insights for players as they engage with the game Own UI feature and screen implementation, from collaborating with stakeholders through integration and launching the completed work into the game Need to Have 4+ years of experience in JavaScript, HTML, CSS development Experience with asynchronous web development (Promises, Async/Await, callbacks) Ability to work with a visual layout tool (Figma, InVision, Framer, Coherent Prysm) Understanding of web components and SPAs (React, Angular, Vue) Experience in profiling and performance optimization Experience working in a collaborative environment with cross disciplinary groups Organizational skills and ability to prioritize efforts Nice To Have Experience with Test Driven Development Experience with Web UI frameworks for Games (Coherent GameFace/GT, EA WebKit) Experience with Unreal Engine and C++ Track record of shipping games and maintaining live games Using web technologies in game development (Unity, HTML5 game engines) Stoic has transitioned to a fully remote, forever remote workforce.
