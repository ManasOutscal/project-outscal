Job Description

We’re working on multi-platform games in Unreal Engine, and looking for an engineer to add to our growing team. Qualifications include: Strong C/C++ programming skills. Expertise in at least one of the core systems in Unreal: graphics, audio, physics, networking, input, user interface, AI, or gameplay programming. Familiar with profiling, debugging, and performance optimization of Unreal games. Shipped at least one game project to a public game store / app store Very comfortable with source Git or Perforce source control for code and assets Experience working with a multidisciplinary team of producers, artists, designers, engineers. We don’t expect our team members to be able to do everything, but bonus points if you’ve: Shipped a mobile, console, or VR game in Unreal. Experience with Jenkins build pipelines or custom build scripting in Python/Groovy/Bash a strong plus. Worked with designers and artists to create custom tools for their workflow. Integrated existing code libraries into Unreal projects. Worked with producers and QA teams to organize your work in Jira or other task / bug tracking systems. To Apply: If you meet most of the criteria, we want to hear from you! We believe in growing and training people. We need you to be able to do the job, but there is still room to add some new skills to your repertoire. If your company/industry calls this role something else, that’s ok too.
