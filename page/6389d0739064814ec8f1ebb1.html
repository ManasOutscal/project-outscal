Job Description

BBTV Interactive is looking for a talented and driven Unity mobile game developer. The ideal candidate loves the idea of working on a small dev team inside a large, well-resourced company. BBTV is the only mobile game developer in the world dedicated to building games for creators on platforms such as TikTok and YouTube. Key Responsibilities Include: Collaborate with producers, designers and artists to ensure the game is delivered on time in alignment with the game vision Own the development of a game from start to finish and through live operations including implementing gameplay, systems, analytics, advertising tech, UI and in app purchases Share a codebase with other developers as needed, delegating responsibilities and reviewing code Assist the producers in milestone planning and prioritization Write clean code with minimal knock on issues, balancing pragmatism against long term systems planning Desired Skills and Experience 5+ yrs of professional gameplay development experience or a strong portfolio of equivalent work Love for creating fun experiences with code Well-versed using Unity and C# to create mobile apps for iOS and Android. Able to break down complicated tasks into discrete solvable components, prioritize what is important, and provide accurate time estimates. You know how to work in a collaborative team environment and navigate the challenges of delivering a great app on time.
