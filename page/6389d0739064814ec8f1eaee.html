Job Title: Junior Unity Programmer Note: Final Designation/Level will be decided based on the relevant experience and interview ratings. Job Summary: As a gameplay programmer, you will be responsible for implementing game features in coordination with design team. You will be responsible for using engine tech features for rapid prototyping. Acting as programming solution provider, you are to explain technical break throughs and limitations to your other team members. Your day today responsibilities include fixing issues in game mechanics, develop new game features. Job Responsibilities: Implement technical solutions. Establish responsible deadlines and personal work plans and manage time effectively. Follow and enhance existing development processes. Co-ordinate with design/art team to create new game features. Debug and fix issues in existing code base. The professional value you bring: 1+ years of Game development experience. Master, Bachelor or equivalent in Computer Science or Engineering (desirable) Strong programming skills in C# , Knowledge of C++ Desirable Strong experience with any commercial game engine (such as Unity 3D, Unreal, Cry engine) Proficiency in game development areas such as Gameplay, UI, Animation etc. Excellent debugging and problem-solving skills Ability to work efficiently with a large, existing code base Knowledge of software development environments and associated tools. (i.e. Visual Studio, Perforce, JIRA, Git) Fluent in English, both verbally and written Great communication skills and a positive attitude Nice to have: Experience with graphics lib Experience with backend /full stack Published at least 1 game on App store/First party The personal value you bring: Curious attitude. Ability to complete duties on time whilst maintaining a high-quality service; Enthusiasm in learning and using new technologies; Good attention to details; Good communication skills; Sense of humor, positive approach, working well within a team. Ubisoft Offers: International collaboration game projects & AAA IP’s. Access to cutting-edge hardware. Experience using hottest internally developed game engines.
